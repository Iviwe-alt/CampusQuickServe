# CampusQuickServe
Java ordering system for Campus QuickServe
This project implements the Campus QuickServe prototype.

Features implemented:
- Customer input using JOptionPane
- Order class with constructor and attributes
- Subtotal and VAT calculations
- 10% discount applied when quantity is greater than 3
- Formatted receipt output

Challenges encountered:
Understanding how to structure the Order class and implementing the discount calculation.
